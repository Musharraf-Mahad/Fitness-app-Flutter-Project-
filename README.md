# Collaboration Guideline

 ## 1. Clone the Repository
They download the project to their computer.
-> git clone https://github.com/Musharraf-Mahad/Fitness-app-Flutter-Project-.git

 ## 2. Navigate to the Project Folder
-> cd repository-name


 ## 3. Create Branch 
Instead of working directly on the main branch.
-> git checkout -b branchName


 ## 4. Make Changes
Edit or add files in the project (code, documentation, etc.).


 # 5. Add the Changes
Stage the modified files.
-> git add .


 ## 6. Commit the Changes
Save the changes with a message.
-> git commit -m "Added new feature"


 ## 7. Push Changes to GitHub
Upload the changes to the repository.
-> git push origin branchName  //  please make sure branchName is to be the branch you created

 ## 8. Create a Pull Request
Go to the repository on GitHub.
Click “Pull Request”.
Request to merge the changes into the main branch.
